# bouncer-admin
The admin interface for https://github.com/mozilla-services/go-bouncer/.

---
#### Requirements
 * docker
 * mysql
---
## How to get started

```sh app.sh```

Run ```docker exec -it local-nazgul pytest``` in a separate terminal to test the code

That's it! Go to [0.0.0.0:5000](0.0.0.0:5000/)
